# University Facility Management System

A Django-based university facility management platform for Lagos State University of Science and Technology.

## Current Scope
- Public landing pages
- Registration and login with email-or-username authentication
- Role-based dashboards
- Complaint submission, assignment, status updates, and comments
- In-app notifications and email notification hooks
- Basic admin reporting

## Local Setup
1. Create a virtual environment.
2. Install dependencies from `requirements.txt`.
3. Copy `.env.example` to `.env` and adjust database settings.
4. Create a MySQL database named `university_facility_management`.
5. Run Django migrations and create a superuser.
6. Start the development server with `python manage.py runserver`.
